# Country Explorer

A simple one-page React app that allows users to search for any country and view details like:

- Flag
- Capital
- Population
- Region

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Mominaaah/Assessment.git
```

2. Open project folder

```bash
cd country-app
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

## API Used

REST Countries API

https://restcountries.com/

## Features

- Search any country
- Error handling
- Loading state
- Input validation
- Responsive simple UI
